== Description ==

Import the Vast Theme demo contents, widgets and customizer settings with one click.

== Installation ==

= Option 1. =

- Upload the entire `vast-demo-import` folder to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Vast Demo Import" —> Click install.
- After plugin is downloaded, click active plugin

= After the plugin active =

- You will find 'Appearance > Vast Demo Import' menu in your WordPress admin panel.
